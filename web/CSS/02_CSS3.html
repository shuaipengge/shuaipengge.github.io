# transition

改变属性触发

transition: width 2s ease；

当width发生变化 ，过渡执行时间，运动的方式，延迟的时间

# animation

可以直接执行 animation: run 2s ease 3s 3 forwards;

动画名 执行两秒 运动方式 延迟的时间 次数 保持最后状态

forwards 设置对象状态为动画结束时的状态

#div1{
    width: 200px;
    height: 200px;
    background: red;
    animation: run 2s ease 3s;
}
@keyframes run {
    0%{
        width: 200px;
    }
    100%{
        width: 400px;
    }
}
