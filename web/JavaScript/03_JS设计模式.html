# 订阅/发布模式 （观察者）

pub/sub

这种模式中，不是一个对象调用另一个对象的方法，而是一个对象点订阅另一个对象的

特定活动并在状态改变后获得通知，订阅者因此也成为观察者，而被观察的对象成为发布者，当发生了一个重要事件的时候 发布者会通知(调用)所有订阅者并且可能经常以事件对象的形式传递消息。
